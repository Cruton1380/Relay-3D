- **Text String Grouping**: Channels grouped by similar topic text strings
- **Left-to-Right Rankings**: Channels ranked by community vote count within each row
- **Live Competition**: Real-time ranking updates as votes change
- **One Vote Per Row**: Users can only vote for one channel candidate within each topic row
- **Geographic Resolution**: Automatic and manual location-based filtering

### Topic Row Voting System
Users participate in two levels of voting:
- **Channel Voting**: Vote for one channel within the topic row (affects ranking)
- **Topic Reliability**: Vote on the relevance and quality of the topic row itself
- **Live Movement**: Channels move left-right in real-time as votes change
- **Face-to-Face Competition**: Channels directly compete for leftward positions

## Channel Type Hierarchy âœ… *Fully Implemented*

### 1. Proximity Channels
**Location-based communities with owner-oriented features**
- Created by users for specific physical locations
- **Signal Control**: Owner can reset via WiFi/Bluetooth to reclaim ownership
- **Owner-Focused**: Enhanced control for location proprietors
- **Proximity Features**: Physical presence affects voting power and access
- **Upgrade Path**: Can evolve to Regional or Global with owner approval

### 2. Regional Channels  
**Community-elected official oriented governance**
- Created by users for geographic regions (cities, states, areas)
- **Community Officials**: Managed by elected representatives
- **Democratic Governance**: Community votes on policies and representatives
- **Regional Features**: Verified residency affects participation rights
- **Origin Tracking**: Can reference original proximity channels they evolved from

### 3. Global Channels
**Founder-oriented universal coordination**
- Created by users for network-wide topics and coordination
- **Founder-Focused**: Enhanced control for global coordination
- **Universal Access**: Available to all verified users worldwide
- **Global Features**: Network-wide policies and development coordination
- **Evolution Chain**: Can track progression from proximity â†’ regional â†’ global
