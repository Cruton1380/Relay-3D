## üîß Technical Implementation

### Loading System

**Backend Service:**
```javascript
// naturalEarthLoader.mjs
class NaturalEarthLoader {
  async getBoundaryGeometry(regionCode, regionType, regionName) {
    // Loads from countries-10m.geojson or provinces-10m.geojson
    // Returns GeoJSON Polygon or MultiPolygon
  }
}
```

**Data Flow:**
1. User requests boundary channel (e.g., "India")
2. `boundaryChannelService.mjs` calls `naturalEarthLoader.getBoundaryGeometry('IND', 'country')`
3. Loader reads from `data/countries-10m.geojson`
4. Returns GeoJSON geometry with 7,737 nodes
5. Frontend renders on Cesium globe

### Geometry Types Supported

**Polygon:**
```json
{
  "type": "Polygon",
  "coordinates": [
    [[lng1, lat1], [lng2, lat2], ..., [lngN, latN]]
  ]
}
```

**MultiPolygon (for countries with islands):**
```json
{
  "type": "MultiPolygon",
  "coordinates": [
    [[[lng1, lat1], [lng2, lat2], ...]],
    [[[lng3, lat3], [lng4, lat4], ...]]
  ]
}
```
