                    THE TWO INDEPENDENT AGES
══════════════════════════════════════════════════════════════


                    MATURITY AXIS (r — radial depth)
                    ↑
                    │   OUTER BARK  (OPEN / ACTIVE — r=1.0)
                    │  ┌────────────────────────────┐
                    │  │  Recently committed data   │
                    │  │  Confidence still settling  │
                    │  ├────────────────────────────┤
                    │  │  Reviewed / stable section  │
                    │  │  High confidence, firm      │
                    │  ├────────────────────────────┤
                    │  │  Archived revision layers   │
                    │  │  CLOSED — migrated inward   │
                    │  └────────────────────────────┘
                    │   CORE (ABSORBED — r=0.0)
                    ▼
───────────────────────────────────────────────────────────────►
                TIME AXIS (l — calendar gravity, always sinking)


  1) TIME-OLD  (l-axis, §3.21)
     • Driven by Earth rotation (globe rotation IS the clock)
     • Always sinks toward trunk / roots
     • Cannot be stopped — this is Gravity

  2) MATURITY-OLD  (r-axis, §3.22)
     • Driven by lifecycle completion
     • Moves inward when obligations are met
     • Independent of calendar age — a 1-day invoice that closes
       immediately migrates inward faster than a 5-year open dispute

══════════════════════════════════════════════════════════════
                    SCAR (RADIAL PROPAGATION)
══════════════════════════════════════════════════════════════

  Revision history layers (cross-section view):

      ────────────────  (latest ring — current timebox)
      ────────────────
      ──────✖─────────  ← revert commit / fraud correction
      ────────────────
      ────────────────  (original ring)

  The scar:
    • Is a dark radial crack, not a red alert badge
    • Propagates outward through every subsequent ring
    • Distorts adjacent ring geometry slightly (stress history)
    • Is permanent — never heals, never fades
    • Is visible in every future cross-section slice
    • Carries: originating timebox, amount, commit ID,
      evidence link count, correction type

  Scar ≠ punishment. Scar = permanent evidence of correction.


══════════════════════════════════════════════════════════════
                    TIMEBOX SLAB (RING) PROPERTIES
══════════════════════════════════════════════════════════════

  Each ring in the cross-section IS a timebox (§4).

  Thick ring   = many commits in that period (busy quarter)
  Thin ring    = few commits (quiet period)
  Dark ring    = high average confidence (well-evidenced)
  Faded ring   = low average confidence (uncertain)
  Warm ring    = high aggregate magnitude
  Cool ring    = low aggregate magnitude

  Reading rings = reading the branch's history
  without opening a single file.


INVARIANT:
  Shape is not decoration.
  Shape is the equation output.
  Two objects with identical data MUST produce identical geometry.
