══════════════════════════════════════════════════════════════
                   GENESIS SEQUENCE  (§80)
══════════════════════════════════════════════════════════════

   Step 1: Account creation (founder)
   Step 2: Tree Zero planted (tree.org.relay-hq)
   Step 3: Archive import (all prior artifacts, timestamps preserved)
   Step 4: Parameter initialization (all extraction rates = 0%)
   Step 5: SCV activation (5 core SCVs)
   Step 6: Disclosure configuration
   Step 7: Genesis seal (Merkle root)
   Step 8: First invitation

   After genesis:
     2D filesystem is a legacy view
     3D tree is the canonical representation
     Both show the same data through different lenses
