══════════════════════════════════════════════════════════════
                    MESH SESSION RULES
══════════════════════════════════════════════════════════════

   • Temporary — dissolves when proximity breaks
   • MeshSession = ephemeral sap-class filament
   • No data mutation (read-only rendering sync)
   • Sap-level sync only (millisecond TTL)
   • Rendering budget shared across devices
   • Any device can leave without breaking others
   • No authentication sharing between devices


══════════════════════════════════════════════════════════════
                    AUDIO MESH
══════════════════════════════════════════════════════════════

   Directional sound field:
      ├── per-device speaker zones
      ├── spatial audio positioning
      └── CrowdAggregate feedback loop
            (arena cheering, ambient data hum)

   Sound follows the same LOD as visuals:
      CELL  = specific filament click/tap
      BRANCH = branch-level ambient tone
      TREE  = tree-level harmonic chord
      GLOBE = civilization hum

   ┌────────────┬───────────┬────────────┐
   │  Phone     │  Laptop   │  VR/Sphere │
   ├────────────┼───────────┼────────────┤
   │  Low-poly  │  Mid-poly │  Full-poly │
   │  2D bark   │  3D bark  │  Filament  │
   │  No sap    │  Sap glow │  Sap flow  │
   │  Basic LOD │  Full LOD │  Full LOD  │
   └────────────┴───────────┴────────────┘

   Feature access never gated by device (§81.9.2).
   Only rendering fidelity varies.
